# Getting started

Admins manage Users, cases and assignments related tasks. 
They are available to select user to view. also available to select case to change.

![support1](img/support1.png)

## Attendance

Click **Attendance**, Type Tutor's name and select the name and clinic, enter date, session and student group then Click **Next**.

![support2](img/support2.png)

Select students and click **Save**.
![support3](img/support3.png)


 Information of the students will be shown on the right hand side of the page, listed **Attendance taken for today (number of students)** 

![support4](img/support4.png)
